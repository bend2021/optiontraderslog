Create Flex Queries and Get Query ID

  1. Login to your Interactive Brokers account

  2. Go to Reports > Flex Queries to create two queries

  3. Click the Activity Flex Query’s plus button to create a new one, Make sure to select the Trades section data

  4. Period select last 365 calendar days, The app will import all trades in this period you selected

    Don’t change the Date Format, and Time Format, and Date/Time Separator or the app won’t work.

  5. Go back to edit the query to see the Query ID that the app needed

    and then

  6. Input this to the All Trades Query ID field of the App settings

  7. Create a new query with less period like Last 7 days with above step and input that Query ID to the New Trades Query ID field

Enable Flex Web Service to Get Access Token

  1. Go to Account Management by click the right top corner user icon

  2. Scroll to the bottom and find Reporting section

  3. If you don’t have token generated, Click Generate New Token to generate one, And Enable the Flex Web Service Status checkbox.

  4. Copy the Current Token value to paste to the App Settings

  5. Click the App right top refresh button to import trades

  6. If the app shows error, Close the popup, and Refresh again, Util it’s successfully imported
